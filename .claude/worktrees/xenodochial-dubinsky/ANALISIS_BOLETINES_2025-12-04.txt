PROBLEMA 1: NO VERIFICA FECHA DEL EMAIL
----------------------------------------
El sistema solo verifica ID, NO fecha.

Si processed_bulletins.txt se borra o corrompe:
  ❌ TODOS los emails de los últimos 30 días se reprocessan
  ❌ Resultado: 209 jobs duplicados (como pasó hoy)

SOLUCIÓN:
```python
# Agregar verificación de fecha
email_date = parse_email_date(message)
if (datetime.now() - email_date).days > 7:
    continue  # Skip emails >7 días
```

PROBLEMA 2: EMAILS NO SE MARCAN EN GMAIL
----------------------------------------
Los emails procesados NO se mueven ni se marcan con label.

Gmail label actual: "JOBS/Inbound"
  ✅ Buenos: Todos en un lugar
  ❌ Malo: No distingue procesados vs nuevos

SOLUCIÓN:
```python
# Después de procesar, agregar label
gmail.users().messages().modify(
    userId='me',
    id=msg_id,
    body={'addLabelIds': ['Label_Processed']}
).execute()
```

PROBLEMA 3: processed_bulletins.txt PUEDE BORRARSE
--------------------------------------------------
Si el archivo se borra (accidental o intencional):
  ❌ Sistema pierde historial
  ❌ Reprocessa TODOS los emails de 30 días

SOLUCIÓN:
- Backup automático del archivo
- O mejor: usar Gmail labels

PROBLEMA 4: NO HAY LÍMITE DE FECHA REAL
---------------------------------------
Query usa `newer_than:30d` pero:
  ❌ NO verifica si email ya fue procesado hace 29 días
  ❌ NO previene reprocessing de boletines viejos

SOLUCIÓN:
- Verificar fecha REAL del email
- Skip si >7 días (los jobs probablemente expirados)

SOLUCIÓN 1: Agregar verificación de FECHA (PRIORITARIO)
--------------------------------------------------------
Archivo: core/automation/job_bulletin_processor.py
Línea: ~290 (después de get message)

AGREGAR:
```python
# Parse email date
email_date_str = None
for header in message['payload']['headers']:
    if header['name'].lower() == 'date':
        email_date_str = header['value']
        break

if email_date_str:
    # Parse date
    from email.utils import parsedate_to_datetime
    email_date = parsedate_to_datetime(email_date_str)
    
    # Skip if older than 7 days
    age_days = (datetime.now(email_date.tzinfo) - email_date).days
    if age_days > 7:
        print(f"   ⏭️  Skipping old email ({age_days} days old)")
        continue
```

BENEFICIO:
  ✅ Emails >7 días automáticamente ignorados
  ✅ Reduce riesgo de duplicados
