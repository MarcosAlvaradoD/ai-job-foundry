## 🚨 QUÉ HACER SI ALGO FALLA

### Error: "No se puede leer columna F"

**Causa:** ApplyURL no está en columna F

**Fix:**
1. Abrir Google Sheets
2. Verificar headers
3. ApplyURL debe estar en columna F
4. Si no, mover columna o modificar código

---

### Error: "No se pueden eliminar emails"

**Causa:** Permisos de Gmail insuficientes

**Fix:**
```powershell
# Borrar token
rm data/credentials/token.json

# Volver a ejecutar (pedirá permisos)
py core/automation/job_bulletin_processor.py
```

---

### Problema: Sigue creando duplicados

**Causa:** URLs tienen formato diferente

**Fix:** Agregar normalización:
```python
# En check_job_exists_in_sheet
job_url = job_url.strip().lower()
existing_urls = [url.strip().lower() for url in existing_urls]
```

---

### Problema: Todos los emails son "old"

**Causa:** Fecha del sistema incorrecta

**Fix:**
```powershell
# Verificar fecha/hora
Get-Date

# Verificar timezone
tzutil /g
```
